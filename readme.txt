Using Frontend Customizer for WooCommerce Plugin the admin can manage various front end settings like changing text, default image of products!

== Description ==

1. Using Frontend Customizer for WooCommerce Plugin, admin can manage settings to change default texts, Default Product Image, pre-define SKU and text for Out of Stock.
2. On installing this plugin, tab named Frontend Customizer for WooCommerce will appear under settings section.
3. Navigating to Frontend Customizer for WooCommerce Tab, admin can change the default text for Add to Cart, Checkout as well as place order.
4. Admin can change default image for products that are added without image.
5. Admin can pre define SKU code for all WooCommerce products. This feature can be enabled/disabled based on admin needs.
6. Admin can also change label name for Out of Stock, globally as well as for each product manually.
7. In order to change label name for Out of Stock manually for each product, it has to be managed under inventory tab found in Product edit page.


== Installation ==

1. Copy the entire /woo-frontend-customizer/ directory into your /wp-content/plugins/ directory.
2. Activate the plugin.
3. New Tab "Frontend Customizer for WooCommerce" will generate.
4. Perform Action.
